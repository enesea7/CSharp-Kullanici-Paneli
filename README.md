# Site Yönetim Sistemi (C# WinForms)

Bu proje C# WinForms ve SQL Server kullanılarak geliştirilmiş kullanıcı yönetim panelidir.

## Özellikler
- Kullanıcı giriş sistemi
- Kullanıcı ekleme
- Kullanıcı güncelleme
- Profil fotoğrafı yükleme
- SQL veritabanı bağlantısı
- Admin kullanıcı paneli

## Kullanılan Teknolojiler
- C#
- .NET WinForms
- SQL Server
- ADO.NET
